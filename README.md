# Calendar Generator

## Descrição
Aplicação para gerar um calendário mensal em PDF, podendo o usuário adicionar eventos/lembretes em cada dias. 
O projeto utiliza uma arquitetura MVC, porém sem framework.

## Tecnologias utilizadas
- PHP
- JavaScript
- Biblioteca mpdf
